Mary Wollstonecraft Shelley - Frankenstein

Project by Elizaveta Cheremisina:

Welcome to the mini digital edition of Chapter 7 of Mary Shelley's Frankenstein. Made as a project for Text as Data II, taught at the University of Antwerp. I enhanced the basic templates by refining transcriptions, encoding text in TEI-XML, and transforming it into HTML with XSLT. This project features the integration of JavaScript functionalities like text highlighting and interactive content display. This work is also hosted on GitHub.
